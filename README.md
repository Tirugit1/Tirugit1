## 🚀 Featured Project

### 🏦 End-to-End CI/CD Pipeline — Banking Application

> A fully automated DevOps pipeline for a high-traffic banking application on AWS + Kubernetes

```
Developer Push
     │
     ▼
 GitHub Repo ──► Jenkins (PollSCM/Webhook)
                      │
              ┌───────┼───────────┐
              ▼       ▼           ▼
           Maven   SonarQube   JUnit Tests
           Build   Analysis    (Quality Gate)
              │
              ▼
        JFrog Artifactory
        (Artifact Storage)
              │
              ▼
    Ansible + Docker Build
    (Image → DockerHub)
              │
              ▼
      Kubernetes Cluster
      (Deployment + Service)
              │
        ┌─────┴──────┐
        ▼            ▼
   Helm Charts    Route53 DNS
   (Versioned)   (Global Exposure)
        │
        ▼
  Prometheus + Grafana
  (Monitoring & Alerts)
```

**What I built:**
- ✅ Automated full pipeline: Code → Build → Test → Scan → Artifact → Deploy
- ✅ Kubernetes multi-node HA deployment with Load Balancer
- ✅ Terraform-provisioned AWS VPC (subnets, IGW, NAT, SGs, NACLs)
- ✅ Prometheus + Grafana monitoring via Helm
- ✅ CloudWatch + SNS alerts for real-time notifications
- ✅ Azure VMs with Load Balancer + Azure Monitor setup

**Tools:** `Git` `Jenkins` `Maven` `JUnit` `SonarQube` `JFrog` `Docker` `Ansible` `Kubernetes` `Helm` `Terraform` `AWS` `Azure` `Prometheus` `Grafana`
